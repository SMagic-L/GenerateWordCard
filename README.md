# GenerateWordCard 生成单词卡

- 一个让你可以手拿把攥地制作纸质单词卡的工具

## 这玩意咋用？

- 确保装有微软的word（程序内需调用VBA的API，所以不支持WPS）。

- 将单词卡的内容以【一行单词，一行翻译】的形式，存储在【UTF-8】格式的txt文件里。

- 打开程序，设置路径后，点击“开始”，生成一个word文档

- 生成word文档后，将其打印出来，并裁剪

- 建议使用裁纸器裁剪，否则你的单词卡可能会犬牙差互（逃
